# 📸 Tè Bé Bổng - Photobooth Pro

Một website Photobooth chuyên nghiệp, phong cách Neon hiện đại, trẻ trung, được xây dựng hoàn toàn bằng **HTML, CSS và JavaScript thuần**.

Giao diện trực quan mô phỏng trải nghiệm chụp hình selfie tại phòng studio thực tế, tích hợp camera trực tiếp, các bộ lọc màu đa dạng và trình chỉnh sửa hình ảnh cao cấp.

---

## ✨ Tính năng nổi bật

### 1. 🖼️ Bố cục đa dạng (Layout Selection)
- **Dải đôi (2-strip):** 2 ảnh kiểu film strip cổ điển.
- **Dải tư (4-strip):** 4 ảnh dọc cao cấp (Được gắn tag ủng hộ tác giả).
- **Lưới 2x2 (2x2-grid):** 4 ảnh dạng polaroid vuông hiện đại.

### 2. 📸 Phòng chụp Selfie đỉnh cao
- Tích hợp camera thời gian thực, tự động căn tỉ lệ chuẩn khung selfie (3:4 hoặc 9:16) mà không bị méo hình.
- Chế độ **Beauty & Smooth Skin** (Làm mịn da, sáng da tự nhiên) trực tiếp trên camera preview trước khi chụp.
- **Hẹn giờ chụp (Timer):** Lựa chọn không hẹn giờ, 3s, 5s hoặc 10s.
- Hỗ trợ tải ảnh lên (`Upload`) trực tiếp từ thiết bị để chỉnh sửa mà không cần dùng camera.

### 3. 🎨 Trình biên tập hình ảnh chuyên nghiệp (Editor Room)
- **Khung viền thông minh:** Các hiệu ứng trang trí nền (Bong bóng, lấp lánh, hoa cúc, bươm bướm, nốt nhạc) chỉ hiển thị ở viền ngoài, cam kết không đè hoặc che khuất chủ thể ảnh.
- **Thay đổi màu nền viền:** Bảng màu neon đa dạng kèm thanh tiêu đề điều khiển trực quan.
- **Nhãn dán (Sticker):** Chèn emoji, nhãn dán vui nhộn dễ dàng kéo thả, phóng to thu nhỏ và xoay 360 độ.
- **Chèn văn bản (Text):** 
  - Kho font chữ phong phú (hơn 13 font từ viết tay, tạp chí đến hiện đại).
  - Tự động lấy lại thông tin văn bản, màu sắc, font chữ cũ khi click chọn lại để chỉnh sửa thời gian thực (Live-Edit).
  - Tùy chọn in ngày tháng (Date stamp) cổ điển.

### 4. 💖 Kêu gọi ủng hộ tùy tâm
- Tích hợp hộp thoại Ủng hộ Tè (`Support Modal`) tinh tế và thân thiện kèm mã QR cá nhân chuyển khoản nhanh khi chọn mẫu Dải Tư.

---

## 🛠️ Công nghệ sử dụng
- **Cốt lõi:** HTML5, CSS3 (Custom Variables, Flexbox, Grid), Vanilla JavaScript (ES6).
- **Icon:** Phosphor Icons qua CDN.
- **Google Fonts:** Tích hợp đa dạng Font chữ hiện đại.
- **Canvas API:** Sử dụng để ghép ảnh, vẽ hiệu ứng nền và xuất ảnh chất lượng cao.

## 🚀 Hướng dẫn khởi chạy nhanh

1. Clone repository về máy:
   ```bash
   git clone git@github.com:caobathien/Photobooth.git
   ```
2. Mở file `index.html` trực tiếp trên trình duyệt hoặc chạy thông qua Live Server trong VS Code để có trải nghiệm tốt nhất với camera!
